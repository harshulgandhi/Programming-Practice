Sample command line input to run this program is as follows:

10
A E
A B 3
B A 3
A D 6
B C 7
D B 5
C D 8
D C 1
D E 9
E D 9
C E 3

First line contains number of connections to be input for stations.
Second line contains Origin ('A' in this case) and Destination ('E' in this case)
Next 10 lines contain connections between different stations with time 
required to travel between these two stations.
